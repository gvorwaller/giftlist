### Key Technical Insight
Codex made excellent strategic cuts (Next Best Action pattern, simplified status labels, 3-item bottom nav, aggressive import deferral) but made a structural misstep by denormalizing occasions onto people. At this scale the performance difference is zero, but normalized tables are cleaner — shared holidays like Christmas should exist once, not be duplicated per person. The muted text color `#5e675f` from Codex mockups fails WCAG AAA (4.6:1 vs required 7:1) on cream backgrounds — darkened to `#3d4a3f`.
